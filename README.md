# WEB103 Prework - *Creatoverse Prework*

Submitted by: **Omar Madjitov**

About this web app: **A React-based web application that allows users to browse, add, edit, and remove content creators. Each creator is displayed with a name, description, and link to their platform, with individual detail pages for more info.**

Time spent: **15** hours
